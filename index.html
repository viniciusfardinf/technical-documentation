Introduction

JavaScript is a versatile programming language.

It runs in the browser and on servers.

console.log("Hello, world!");
  • Client-side
  • Server-side
Syntax

JavaScript syntax is the set of rules.

It includes how programs are constructed.

let x = 5;
  • Statements end with semicolons
  • Case-sensitive
Variables

Variables store data values.

You can declare them with var, let, or const.

const pi = 3.14;
  • var (old)
  • let (block-scoped)
  • const (read-only)
Functions

Functions perform tasks or calculate values.

They can be declared or expressed as values.

function add(a, b) { return a + b; }
  • Reusable
  • Modular
Loops

Loops execute a block of code repeatedly.

Common loop types include for and while.

for (let i = 0; i < 5; i++) { console.log(i); }
  • for loop
  • while loop
  • do...while loop
